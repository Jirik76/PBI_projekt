# Popis projektu – Power BI report

Cílem projektu je vizualizace vývoje průměrných mezd a cen vybraných potravin v České republice a jejich vzájemné porovnání v čase. Report vychází z připravených datových podkladů a navazuje na analytickou část předchozího projektu.

Power BI report je rozdělen do tří tematických stránek – **Mzdy, Ceny potravin a Porovnání mezd a cen**. Uživatel může pomocí průřezů (slicerů) filtrovat jednotlivé roky, přičemž všechny vizuály se dynamicky přepočítávají podle zvoleného období.

V reportu jsou využity různé typy vizualizací (karty, spojnicové a sloupcové grafy), vlastní míry vytvořené v jazyce DAX a vzájemně propojené datové tabulky. Navigace mezi stránkami je řešena pomocí interaktivních tlačítek. Výsledkem je přehledný a interaktivní report umožňující sledovat dlouhodobé trendy a porovnávat vývoj mezd a cen potravin.
